## 数据来源与更新节奏

- **实时层**：[Web3Watch-HK](https://github.com/Beltran12138/Web3Watch-HK) 每日抓取 25 个 HK Web3 来源，高 alpha 事件自动推送
- **知识层（本 Repo）**：每周人工汇总，AI 辅助提炼
- **覆盖范围**：香港持牌交易所、SFC 监管动态、RWA/代币化项目、AI Agent 交易浪潮

## 设计理念

本 Wiki 采用 [Karpathy LLM Wiki Pattern](https://github.com/karpathy/llm.c)：

```
实时数据（Raw Sources）
       ↓
  知识库（本 Repo）       ← 人工 + AI 维护
       ↓
  AI 日报提示词注入
       ↓
  战略建议（> 事件摘要）
```

Wiki 文件为结构化 Markdown，可直接被 AI Agent 消费（机读友好），同时对人类读者友好。
