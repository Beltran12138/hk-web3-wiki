# OSL 竞品分析

## 基本数据

| 指标 | 数据 |
|------|------|
| 定位 | 全球首家双重持牌（Type1 + VATP），MSCI 成分股 |
| OTC 最低门槛 | USD $100,000 |
| 保险覆盖 | 10 亿美元（全球记录） |
| Omnibus Pro | 已接入 50+ 家中小券商，主导 B 端市场 |

## 近期重要动作（2025-2026）

- **RLUSD 上线**：香港首批合规 Ripple 稳定币
- **USDGO**：B2B 自有结算稳定币
- **iRFQ SaaS**：商业化技术输出
- **OpenEden + Plume 合作**：RWA 链上资产接入
- **OSL Tokenworks**：代币化发行平台
- **EXCB25**：债券代币化产品

## 合规结构关键点（Omnibus Pro）

- 双层 KYC：OSL 不直接持有终端用户信息，合作方负责底层 KYC
- 法币出入金：用户须从本人名下银行账户充值，企业账户不可代转
- Travel Rule 强制执行，平台不能在不同用户间互转
- 终端用户无独立链上地址（账本权益形式）

## 弱点 / 市场空白

- 门槛最高（$10 万），中小客户市场空白
- Omnibus Pro 集成复杂，**1,000+ 家中小券商仍观望**
- 代币覆盖较窄（合规审核严格）
- iRFQ SaaS 白标业务与自身 B 端客户存在竞争关系
