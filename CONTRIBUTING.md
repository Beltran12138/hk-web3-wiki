# 贡献指南

感谢你对 HK Web3 Market Intel Wiki 的贡献！

## 贡献方式

### 1. 更新现有页面

发现过时信息或有新数据？直接提 PR：
- 每个 Markdown 文件顶部有 `last_updated` 字段，请同步更新
- 竞品数据注明来源（官网/公告/媒体报道）
- 监管内容注明 SFC 原文链接

### 2. 新增竞品页面

复制 `竞品分析/_template.md`，填写以下必填字段：

```markdown
---
title: 竞品名称
category: competitor
last_updated: YYYY-MM-DD
sources: [官网, SFC公告, ...]
---
```

**必填章节：** 基本信息、核心产品、近期重要动态、弱点（我司机会）

### 3. 新增监管/趋势页面

同理，复制对应模板，保持章节结构一致以便 AI Agent 提取。

## 内容规范

- **客观中性**：竞品分析基于公开信息，不做主观贬低
- **时效标注**：数据有效期超过 3 个月请标注 `[需更新]`
- **来源追溯**：关键数据（用户数、AUM、牌照状态）须附来源
- **机读友好**：使用标准 Markdown 标题层级（`##` 一级章节），避免复杂表格嵌套

## 提交格式

```
docs(竞品-HashKey): 更新 2026-04 GUSDT 稳定币动态
docs(监管-SFC): 补充 2026-03 VATP 牌照第二批名单
```

## 不接受的内容

- 未经核实的市场传言
- 含有价格预测或投资建议
- 内部机密或非公开信息
